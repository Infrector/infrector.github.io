Buffon Needle Simulation in Matlab

The Buffon's needle Monte Carlo simulation approximates π by repeatedly dropping needles on a surface with parallel lines spaced apart. The process involves:
Randomly generating positions and angles for the needles.
Determining if each needle intersects any of the lines based on its position and angle.
Calculating the ratio of intersecting needles to the total number of needles.
Using the probability of intersection derived from the geometry of the setup, this ratio helps estimate π. The accuracy of the approximation improves with an increasing number of needle drops.

The simulation uses random values for place and angles but in the end the value will always be close to π.

Minesweeper in C#

The Minesweeper app in C# is a grid-based puzzle game where players reveal cells while avoiding hidden mines. The game provides a configurable grid size and mine count, with mine positions randomized for each new game. Players left-click to reveal cells, showing either a mine or a number indicating adjacent mines. Right-clicking flags cells suspected of containing mines. The game is won by revealing all non-mine cells and lost instantly when a mine is clicked. The user interface is built using Windows Forms, with each cell represented by a button.

Blender Render

My Blender project features a glass cup filled with a blue fluid, showcasing realistic glass properties with varied settings.
